动态执行 JS

发表于 2023-11-27 分类于 javascript

如何把一个字符串，当成一个 JS 代码来执行？

eval

eval() 函数会将传入的字符串当做 JavaScript 代码进行执行,它会返回字符串中代码的返回值。如果返回值为空，则返回 undefined。

setTimeout

setTimeout() 中的 code 参数是一个可选语法，它允许你包含在定时器到期后编译和执行的字符串而非函数。
